# IBM Generative AI-Powered Applications Projects

Portfolio of 6 hands-on projects completed as part of the **IBM Generative AI Engineering Professional Certificate - Course 6: Building Generative AI-Powered Applications with Python**.

This repository demonstrates practical skills in building intelligent chatbots and AI applications using Large Language Models (LLMs), RAG, voice technologies, and modern web frameworks.

## Projects

| Project | Description | Key Technologies |
|---------|-------------|------------------|
| [Project 1: Image Captioning](project_1-image-captioning) | AI tool that generates meaningful descriptions for uploaded photos | LLMs (GPT-3 / Llama 2), Hugging Face / watsonx, Gradio |
| [Project 2: Web Chatbot](project_2-web-chatbot) | Interactive chatbot similar to ChatGPT with web interface | Flask, HTML/CSS/JS, LLM integration |
| [Project 3: Voice Assistant](project_3-voice-assistant) | Voice-enabled assistant (speech in → voice out) | IBM Watson STT/TTS, Python |
| [Project 4: Meeting Transcriber](project_4-meeting-transcriber) | Transcribes meetings and produces concise summaries | Speech-to-Text, LLM summarization |
| [Project 5: Generative AI with RAG](project_5-generative-ai-rag) | Upload PDF → ask questions and extract information | RAG, LangChain, PDF parsing |
| [Project 6: Voice Translator](project_6-voice-translator) | Real-time speech translation assistant | STT + TTS + Translation LLM |

## Tech Stack
- **Python** (core)
- **LLMs**: GPT-3, Llama 2 (via IBM watsonx & Hugging Face)
- **Frameworks**: Flask, Gradio
- **Voice**: IBM Watson Speech-to-Text (STT) & Text-to-Speech (TTS)
- **Orchestration**: LangChain, RAG
- **Frontend**: HTML, CSS, JavaScript

## Learning Objectives Demonstrated
- Building end-to-end generative AI applications
- Integrating LLMs with real-world data sources and interfaces
- Implementing Retrieval Augmented Generation (RAG)
- Voice-enabled AI interactions
- Production-ready project structure and documentation
